# Frontend Application Shell

FMS-0007 Phase 2 creates the initial frontend navigation, layout components, route structure, and mock data views.

## Scope

- Application shell
- Sidebar and top bar navigation
- Dashboard and operational route placeholders
- Building list, detail, and passport pages
- Reusable display components
- Mock data only

## Out of Scope

- Backend integration
- Authentication logic
- Business module workflows
- Real data persistence
