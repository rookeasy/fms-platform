# Fuzion Whole-F Lifecycle System

Fuzion is the primary brand.

The product is Fuzion Operating Platform™.

The workspace is Mission Control™.

Preferred browser title:

`Mission Control™ | Fuzion Operating Platform`

## Brand Hierarchy

- Company: Fuzion Tech Inc.
- Primary brand: Fuzion
- Product: Fuzion Operating Platform™
- Workspace: Mission Control™
- Lifecycle framework: BUILD • ADVISE • PROTECT
- Promise: Buildings Under Protection™

FOP may remain a compact internal abbreviation where space is limited. It should not replace Fuzion as the primary customer-facing brand.

Do not use FTP as an acronym.

Do not display `F Mission Control`.

## Static Platform F

The static F represents the platform.

Rules:

- whole F
- single colour
- deep navy
- flat
- no gradient
- no halo
- no embedded labels
- no segmented colour arms
- no lifecycle animation

Use the static F for the sidebar, favicon, app icon, top-level platform shell, navigation, compact headers, and product documentation.

## Dynamic Building F

The dynamic F represents one building.

The entire F changes colour as the building progresses. It never displays separate coloured arms, embedded BUILD / ADVISE / PROTECT labels, or simultaneous multi-colour lifecycle fills.

Lifecycle states:

- BUILD: whole F in electric royal blue `#126BFF`
- ADVISE: whole F in refined platinum silver `#BFC5CE`
- PROTECT: whole F in pastel vibrant green `#86EE62`
- PROTECTED: green F plus restrained coral halo

Colour is never the only signal. The visual state must also expose text such as BUILD, ADVISE, PROTECT, or PROTECTED through visible labels or accessible labels.

## Halo Rule

The halo is authoritative.

The frontend may show the coral halo only when backend protected-state data confirms:

- `protected_state_status = approved`
- `halo_eligible = true`
- not suspended
- not revoked

Never infer PROTECTED from completion percentage alone.

## Lifecycle Mapping

Use existing backend status data.

BUILD:

- active construction
- Building Registered
- early project lifecycle

ADVISE:

- documents imported
- commissioning
- evidence review
- closeout review
- Passport preparation

PROTECT:

- Ready for Passport
- Ready for Protected-State Review
- ITM/protection transition underway
- not yet formally approved

PROTECTED:

- backend-authoritative protected-state approval
- halo eligible

If backend data is unavailable, render neutral or no halo.

## Prohibited Treatments

- segmented multi-colour F
- top/middle/stem arms coloured separately
- rainbow gradients
- BMW-like blue/red styling
- labels embedded inside the F
- halo on static platform F
- halo inferred by the frontend alone
- `F Mission Control`
- dominant customer-facing `FOP™` where Fuzion should lead
